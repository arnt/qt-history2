The Connection Viewer Dialog (View Connections)

Figure 9-3. The connection viewer dialog

This dialog is invoked by selecting Edit|Connections from the menu bar. It shows the connections between signals and slots that are in place for the currently active form. The central, unlabeled list box shows the outgoing GUI element, the signal name, the incoming GUI element, and the slot name for each connection.

Edit

Brings up the Connection Editor dialog (see Section 7.2), which lets you edit all connections between the outgoing and incoming GUI elements in the currently selected connection.

Disconnect

Removes the currently selected connection.
