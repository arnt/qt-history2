The Preview Menu

Preview Form

Displays the currently active form as it will look in your program in the default style of your platform (Motif style on Unix systems, Windows style on Windows systems).

Preview Form in Motif Style

Displays the currently active form as it will look in your program in Motif style.

Preview Form in Motif Plus Style

Displays the currently active form as it will look in your program in Motif Plus style (gtk-like).

Preview Form in Windows Style

Displays the currently active form as it will look in your program in Windows style.

Preview Form in Platinum Style

Displays the currently active form as it will look in your program in Platinum style.

Preview Form in CDE Style

Displays the currently active form as it will look in your program in CDE style.

Preview Form in SGI Style

Displays the currently active form as it will look in your program in SGI style.
