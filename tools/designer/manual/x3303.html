The Edit Multiline Edit Dialog

Figure 9-8. The multiline editor

This dialog lets you edit the text to appear in a multiline edit. You reach it by selecting Edit from the context menu of a multiline edit.

This dialog is pretty simple: just enter the text you want displayed in the multiline edit in the Text field.
