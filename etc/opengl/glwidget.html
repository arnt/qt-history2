Detailed Description

The GLWidget is an OpenGL wrapper class.

To create your own OpenGL widget, make a subclass of GLWidget and reimplement the virtual functions paintGL() and resizeGL(). Study the Nurb class for an example.

The GLWidget is a very thin wrapper and it probably lacks functionality. Please give feedback to hanord@troll.no.
