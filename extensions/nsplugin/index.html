Qt-based LiveConnect Plugins for Netscape and Explorer

The Netscape and Explorer WWW browsers allow for Plugin software to be added to support information presentation to supplement normal WWW browsing. The Plugin interface is described in Netscape's handbook.

The Qt-based Netscape Plugin software makes it easy to write such plugins such that they can be used on both Unix/Linux and Windows/95/NT.

How-to

First, build the Netscape Plugin extension library, found in the extensions/nsplugin/src directory of your Qt distribution. Then, to find out how to write Qt-based Netscape Plugins, read the plugin class documentation, and examine the example plugins listed below. Note that under Windows, plugins must be named npsomething.dll, or the browser will ignore them.

Examples

The following example programs are available to demonstrate the use of the Qt GUI toolkit for making Netscape Plugins.
  • Grapher - graph data without resorting to GIFs
  • QtImage - plugin for Qt-supported image formats, including PNG.
