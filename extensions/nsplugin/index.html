Qt-based LiveConnect Plugins for Netscape

The Netscape WWW browser allows for Plugins software to be added to support information presentation to supplement normal WWW browsing. The Plugin interface is described fully here.

The Qt-based Netscape Plugin software makes it easy to write such plugins such that they can be used on both Unix/Linux and Windows/95/NT.

How-to

To find out how to write Qt-based Netscape Plugins, read the plugin class documentation.

Examples

The following example programs are available to demonstrate the use of the Qt GUI toolkit for making Plugins for Netscape.
  • Grapher - graph data without resorting to GIFs
  • QtImage - plugin for Qt-supported image formats, including PNG.
