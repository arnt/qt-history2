Qt-based Plugins - QtImage

This sample plug-in demonstrates Qt used for developing Netscape Plug-ins. For this to work, you must either already have a PNG plugin (in which case this example is not very interesting), or you must compile and install the plugin code - type "make install" in the extensions/nsplugin/examples/qtimage directory of your Qt installation.

This plugin displays a PNG format image.

With the simplicity and cross-platform nature of Qt-based plugins such as this, any new image format can be supported by adding new image I/O handlers to the Qt library, thus providing the functionality in both applications and WWW plugins from the same code.


This plugin supports transparency and scaling of the image, just like GIFs in most WWW browsers.
