Rusty's Late

by John VanSickle
ANIMATION DESCRIPTION:

Rusty the Robot is late to his weekly three-way chess game.

DESCRIPTION OF HOW THIS ANIMATION WAS CREATED:

I typed out a bunch of scene
code with Notepad, and I also used the file writing capabilities of POV 3.1 to create some mesh files. The trees, the road, the tractor tread, and the crystal shell are meshes. Everything else is CSG. I didn't include the scene code because there is over 270K of it.
