How to use this Example

There are three ways to get help in this example:

  • Press F1. Depending where your mouse position you get either the help for the item under the mouse cursor dispayed or the general help for this example.
  • Press the right mouse button to invoke a context menu, then click 'Help'. The displayed help page is choose as decribed above.
  • Use the buttons to control Qt Assistant directly. For a further explaination see below.



Install Example Documentation

Pressing this button will install the profile for the HelpDemo application. When the profile is installed and Qt Assistant is later run with the button Run Qt Assistant customized for HelpDemo checked, Qt Assistant will have a different application icon, customized toolbar, a special about option etc.







Open Assistant Button

Click this button to open Qt Assistant. If it is already open nothing will happen. The displayed page is the last displayed one.







Close Assistant Button

Click this button to close Qt Assistant. If it is not running nothing will happen.







Run Qt Assistant customized for HelpDemo

Check this item in order to run Qt Assistant as a customized help tool for the HelpDemo example. Checking or unchecking has no effect if Qt Assistant is already running. To apply changes you have to restart Qt Assistant.

NOTE: Before you can use this option, the profile has to be installed. This is done by pressing the Install Example Documentation button.







Hide Sidebar on Startup

Check this item to hide the sidebar in Qt Assistant on the next startup. Checking or unchecking has no effect if Qt Assistant is already running. To apply changes you have to restart Qt Assistant.







Line Edit

Enter here the file name of a help page you want to get displayed. The path could be either absolute or relative.







Display Button

Click this button in order to get the help page specified in the file name line edit displayed. If Qt Assistant is currently not opened, it will be opened automatically.







Close Button

Click this button to quit this example application.
